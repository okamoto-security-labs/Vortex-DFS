Core guarantees

Three laws.
No exceptions.

Every packet that enters Vortex is evaluated against deterministic mathematical invariants — not patterns, not models, not predictions.

⚛

Post-quantum by design

Signatures based on Learning With Errors — the mathematical foundation of NIST's 2024 post-quantum standards. A quantum computer does not break this.

∫

Physics-bound trust

Trust scores derived from distance and entropy metrics, evaluated against deterministic thresholds. Not a model. Not a heuristic. Pure mathematics.

⊢

Zero ambiguity

Every packet is Accept or Reject with a typed, auditable reason. No silent failures. No undefined behavior. No gray area an attacker can exploit.

Architecture

Four layers.
One decision.

A packet only reaches the next layer if it fully satisfies the current one. Each layer has one job and one failure mode.

01

Protocol protocol.rs

Binary parsing via explicit byte reads — no pointer casting, no unsafe. CRC-32 validation over header and payload before any processing.

02

Authentication vortex_guard.rs

HMAC-SHA256 with constant-time comparison. Body size enforced at 1MB. Session IDs sanitized before logging. Key loaded from environment — never hardcoded.

03

Signature signer_lwe.rs

Fiat-Shamir over LWE lattice. Commitment binding via hash — forging a signature without the secret key requires solving a problem quantum computers cannot.

04

Trust evaluation engine.rs + pqc_core.rs

Distance and entropy metrics evaluated against physics-derived thresholds. Returns a typed TrustBand: HighTrust, Operational, Fragile, or Critical.

Quick start

Five lines
to post-quantum security.

Add Vortex to your Rust project and start evaluating packets with full post-quantum authentication.

main.rs 
// 1. Generate a keypair
let (sk, pk) = keygen(seed);
let gate = VortexGate::new(pk.clone());

// 2. Sign and send a packet
let payload = build_telemetry(distance, entropy);
let sig = sk.sign(&payload, &pk, nonce);

// 3. Evaluate on receipt
match gate.process_packet(&raw, &sig) {
    TrustState::HighTrust               => { /* proceed */ }
    TrustState::Operational             => { /* proceed with monitoring */ }
    TrustState::Fragile                 => { /* degrade gracefully */ }
    TrustState::RejectedSignature       => { /* block — forged packet */ }
    TrustState::RejectedProtocol(why)   => { /* block — malformed */ }
    TrustState::RejectedBounds          => { /* block — invalid metrics */ }
}
